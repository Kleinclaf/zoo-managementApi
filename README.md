# Zoo Management API (.NET 9 Minimal API)

## Project Summary
Zoo Management API is a RESTful backend application designed to manage the basic operations of a zoo, including animals, enclosures, and responsible staff members.
The project is developed using .NET 9 Minimal API and follows a layered architecture approach.

## Project Goals
- Practice building RESTful APIs using .NET 9 Minimal API
- Apply layered architecture principles
- Design clean and resource-oriented REST endpoints
- Use DTOs instead of exposing domain entities directly
- Improve code readability and maintainability
